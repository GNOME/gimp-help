7.10. Discard Text Information

This command belongs to a group of Text commands displayed only if a text layer is present.

Figure 16.107.  The Discard Text command among text commands in the Layer menu


When you add text to an image, GIMP adds specific information. This command lets you discard this information, transforming the current text layer into a normal bitmap layer. The reason to do that is not evident.

Note that this transformation of text into bitmap is automatically performed when you apply a graphic operation to the text layer. You can get text information back by undoing the operation which modified the text.

7.10.1. Activating the Command

  • You can access this command from the image menubar through Layer → Discard Text Information.
