7.8. Delete Layer

The Delete Layer command deletes the current layer from the image.

7.8.1. Activate the Command

  • You can access this command from the image menubar through Layer → Delete Layer.

  • In addition, at the Layer Dialog, you can access it through Delete Layer of its context pop-up menu, or clicking on the icon button on the bottom of this dialog.
