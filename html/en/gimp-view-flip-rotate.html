5.5. Flip and Rotate (0°)

You can access this command from the image menubar through View → Flip and Rotate (0°),

This command opens a submenu:

Figure 16.50.  The “Flip and Rotate” submenu


These commands act on the view, not on the image. To act on the image, go to: Image Transform.

These commands are self-explanatory. Please note the “!” shortcut for the Reset Flip and Rotate command, which restores the image in its original state.

The Other… command opens a dialog that allows you to set a precise angle for the rotation of the view:

Figure 16.51.  The “Select Rotation Angle” dialog
