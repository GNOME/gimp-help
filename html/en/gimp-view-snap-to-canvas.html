5.18. Snap to Canvas

If this option is enabled, when you move a selection or a layer, they appear to pull on the canvas edges when it approaches. This is useful for accurate placement of image elements.

Note

Canvas edges are usually mingled with image edges: the canvas has, then, the same size as the image. But you can change canvas size in Image → Canvas Size.

5.18.1. Activating the Command

  • You can access this command from the image menubar through View → Snap to Canvas.
