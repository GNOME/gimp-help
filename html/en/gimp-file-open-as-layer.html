2.5. Open as Layers…

The Open Image as layers command opens the Open Image dialog. The layers of the selected file are added to the current image as the top layers in the stack.

2.5.1. Activate Command

  • You can access this command from the image menubar through File → Open as layers…,

  • or by using the keyboard shortcut Ctrl+Alt+O.
