8.32. Sepia

8.32.1. Overview

Figure 16.200. Example for the “Sepia” filter

Original image

“Sepia” filter applied


The Sepia filter imitates sepia toning to produce a black-and-white picture with a warm brown-ish tone.

8.32.2. Activate the filter

You can find this command through Colors → Desaturate → Sepia….

8.32.3. Options

Figure 16.201. “Sepia” options


Effect strength

This handle lets you control how much the effect blends into the original image. 0.000 is no effect at all, 1.000 is completely replacing the image with its sepia-toned version.

sRGB

This checkbox allows you processing the image in either sRGB gamma-corrected or linear color space.
