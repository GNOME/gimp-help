14.8. Simplex Noise

14.8.1. Overview

This filter generates a noise texture using the Simplex Noise algorithm. Results don't depend on the image you opened

Filter applied with default options

14.8.2. Activating the filter

This filter is found in the image window menu under Filters → Render → Noise → Simplex Noise….

14.8.3. Options

Figure 17.333. “Simplex Noise” filter options


Presets, Preview, Split view, Use selection as input
Note

These options are described in Section 2, “Common Features”.

Scale

Scale of the noise function: works like a zoom.

Iterations

The number of noise octaves.

Random seed, New seed

To vary noise.
