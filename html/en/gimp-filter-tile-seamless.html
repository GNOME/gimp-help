13.10. Tile Seamless

13.10.1. Overview

Figure 17.299. An example of Tile Seamless.

Original

Tile Seamless applied


This filter modifies the image for tiling by creating seamless edges. Such an image can be used as a pattern for a web-page. This filter has no option, and result may need correction.

13.10.2. Activation

You can find this filter through Filters → Map → Tile Seamless…
