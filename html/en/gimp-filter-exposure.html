8.7. Exposure

This filter is used for lightning shadows and midtones of an image without blowing the highlights. A detailed tutorial can be found at Appendix D, Tone Mapping and Shadow Recovery Using GIMP’s ‘Colors/Exposure’ .

8.7.1. Activating the Command

You can get to the Exposure dialog through: Colors → Exposure…

8.7.2. Options

Figure 16.148. Exposure Options


Presets

“Presets” are a common feature for several Colors command. You can find its description in Section 8.1, “Introduction to the “Colors” Menu”.

Black level

This option is used to dispense with "useless" shadows and highlight information.

Exposure

This option is used to add one or more stops of exposure compensation to the image.

Preview and Split view

These are common features described in Section 8.1, “Introduction to the “Colors” Menu”.
