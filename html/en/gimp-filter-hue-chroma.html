8.4. Hue Chroma

The Hue-Chroma filter is another way to modify saturation. Clear explanations about chroma and saturation can be found at https://ninedegreesbelow.com/photography/changing-saturation-using-lch-chroma.html .

8.4.1. Activating the Command

You can get to the Hue-chroma dialog through: Colors → Hue-Chroma…

8.4.2. Options

Figure 16.144. Hue-Chroma Options


Presets

“Presets” are a common feature for several Colors command. You can find its description in Section 8.1, “Introduction to the “Colors” Menu”.

Hue

The slider and the small arrow buttons allow you to select a hue in the color circle (-180°, 180°).

Chroma

“Chroma” is the purity of the color. Moving this slider makes colors more or less pure.

Lightness

This slider and the small arrow buttons allow you to select a value (luminosity): -100, 100.

Preview and Split view

These are common features described in Section 8.1, “Introduction to the “Colors” Menu”.
