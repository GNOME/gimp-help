3.5. File Manager

If you have associated an image file type with GIMP, either when you installed GIMP or later, then you can navigate to the file using a file manager (such as Nautilus or Konqueror in Linux, or Windows Explorer in Windows), and once you have found it, double-click on the file. If properly configured, the image will open in GIMP.
