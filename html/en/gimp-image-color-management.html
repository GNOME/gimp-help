6.8. Color Management

The Color Management submenu contains commands which let you change the ICC color profile associated with an image and also let you save the associated ICC color profile to disk. There are five options in the Color Management submenu.

Figure 16.69.  The “Color Management” submenu of the “Image” menu


6.8.1. Activating the Color Management Submenu

You can access this command from the image menu bar through Image → Color Management.

6.8.2. The Contents of the “Color Management” Submenu

  • Enable Color Management

  • Assign Color Profile

  • Convert to Color Profile

  • Discard Color Profile

  • Save Color Profile to File
