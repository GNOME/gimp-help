7.48. Arbitrary Rotation

The Arbitrary Rotation command rotates a layer by a specified angle. It is an alternate way of accessing the Rotate tool. See the section about that tool for more information.

7.48.1. Activating the Command

  • You can access this command from the image menubar through Layer → Transform → Arbitrary Rotation,

  • or by using the keyboard shortcut Shift+R.
